##  💻 Guia para Configuração do Ambiente de Desenvolvimento

### 1. Clone o Repositório de Documentação
Antes de iniciar, você precisa clonar o projeto e acessar a branch de desenvolvimento:

```shell
git clone https://github.com/mdsreq-fga-unb/2024.2-T01-FamintosBurguer.git
```

```shell
git checkout develop
``` 

### 2. Instale as Dependências do Projeto

#### 2.1. Instalando usando o NPM:
O Projeto usa dependências e ferramentas específicas para seu funcionamento, ferramentas estas que já estão presentes no arquivo "package.json", restando ao usuário apenas instalar com apenas um comando.

#### 2.2. Acesse o terminal e digite o seguinte comando:

```shell
npm install
```

### 3. Executando o Projeto:
Após instalar as dependências do projeto, siga os seguintes passos para inicializar o projeto:

### 3.1. Acesse o diretório do projeto

```shell
cd .\aplicacao\
```

### 3.2. Execute o comando

```shell
npm run dev 
```

*Caso ocorra algum erro durante a instalação, por favor, sinta-se a vontade para abrir uma issue para resolução do problema.*
